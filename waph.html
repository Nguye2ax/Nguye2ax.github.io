Course Overview

Web Application Programming and Hacking is a course where I learn how web applications work and how to secure them. The course covers front-end development, back-end basics, web APIs, and common web security problems.

Labs

Lab 0 - Development Environment

In Lab 0, I set up my development environment using Ubuntu, Apache, Git, GitHub, and other tools needed for the course.

Lab 1 - Foundations of the Web

In Lab 1, I practiced HTTP requests, CGI, PHP, and how a browser communicates with a server. This helped me understand what happens behind the scenes when a webpage loads.

Lab 2 - Front-end Web Development

In Lab 2, I worked with HTML, CSS, JavaScript, Ajax, jQuery, and public Web APIs. I learned how a webpage can update without fully reloading and how APIs return data like JSON.

Hackathons

Hackathon 1 - Cross-Site Scripting

In Hackathon 1, I practiced reflected XSS attacks in the course environment and then fixed my own vulnerable code. I learned that output encoding is important because user input should not be treated as executable JavaScript.

Individual Project 1

Individual Project 1 is my GitHub Pages professional profile website. For this project, I created a homepage, added this WAPH course page, used Bootstrap for styling, added JavaScript and jQuery, used Vue.js as another JavaScript library, added cookies to remember the visitor, included a page tracker, and integrated two public APIs.
